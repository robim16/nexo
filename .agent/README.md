# 🧠 Skills de Nexo — Catálogo

Colección de skills de arquitectura empresarial para aplicaciones Vue 3 + Firebase.
Cada skill encapsula conocimiento experto sobre un dominio específico del desarrollo.

## 📦 Skills Disponibles

| Skill | Carpeta | Descripción |
|-------|---------|-------------|
| 🏗️ Arquitectura Senior | `skills/vue-arquitectura-senior/` | Arquitectura Limpia, principios SOLID, estructura de carpetas e inyección de dependencias para apps Vue empresariales |
| 🔥 Firebase Integración Avanzada | `skills/firebase-integracion-avanzada/` | Repositorios tipados, tiempo real, autenticación, Storage y optimizaciones en Firebase |
| 🎨 Patrones de Diseño Vue | `skills/vue-patrones-de-diseno/` | Factory, Builder, Singleton, Adapter, Decorator, Composite, Strategy, Observer y más |
| ⚡ Optimización de Rendimiento | `skills/vue-optimizacion-rendimiento/` | Code splitting, scroll virtual, lazy loading, memoización, Web Vitals |
| 🧪 Pruebas Senior | `skills/vue-pruebas-senior/` | Vitest, Vue Test Utils, Playwright, mocks, TDD, cobertura |
| 🔷 TypeScript Avanzado | `skills/vue-typescript-avanzado/` | Tipos genéricos, Branded Types, type guards, tipos utilitarios |

## 🚀 Proyecto: Nexo

**Nexo** es una red social moderna construida con:
- Vue 3 + Composition API + TypeScript strict
- Firebase (Auth + Firestore + Storage)
- Arquitectura Limpia por capas
- Pinia para estado global
- Vite como bundler

## 📐 Regla de Uso

Los skills se activan automáticamente según el contexto de cada tarea.
Consulta el `SKILL.md` de cada carpeta para ver sus triggers y contenido completo.
